Identificador: UC-021

Nome: Verificar Funcionários

Ator Principal: Atendente

Descrição: Apresenta ao atendente o submenu de opções para gerenciamento dos funcionários da academia.

Pré-condições: Atendente logado e no Menu Principal.

Pós-condições: Sistema exibe as opções de operação para funcionários.

Fluxo Principal:

    [IN] Atendente seleciona a opção "Verificar Funcionários" no Menu Principal.

[OUT] Sistema apresenta o submenu com as opções: [1] Listar, [2] Apagar, [3] Ver, [4] Alterar, [Sair] Voltar.

[IN] Atendente seleciona uma das opções disponíveis.

[SYSTEM] Sistema redireciona para o caso de uso correspondente à opção escolhida.
