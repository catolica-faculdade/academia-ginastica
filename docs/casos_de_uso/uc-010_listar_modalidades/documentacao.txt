Identificador: UC-010

Nome: Listar Modalidades

Ator Principal: Atendente

Descrição: Exibe uma relação completa de todas as modalidades esportivas atualmente cadastradas no sistema.

Pré-condições: Atendente logado no menu de modalidades.

Pós-condições: Nenhuma alteração de dados.

Fluxo Principal:

    [IN] Atendente seleciona a opção "Listar Modalidades" no menu.

    [SYSTEM] Sistema consulta o banco de dados por todas as modalidades ativas.

    [OUT] Sistema exibe a lista na tela (ex: ID e Nome de cada modalidade).

    [OUT] Sistema exibe mensagem "Pressione qualquer tecla para voltar...".

    [IN] Atendente pressiona uma tecla.

    [OUT] Sistema retorna ao menu de modalidades.

Fluxo Alternativo 2a: Lista Vazia

    [SYSTEM] Sistema verifica que não há nenhuma modalidade cadastrada.

[OUT] Sistema exibe mensagem "Nenhuma modalidade encontrada." e questiona se deseja cadastrar uma agora.

[IN] Atendente pode optar por "Sim" (redireciona para UC-006) ou "Não" (volta ao menu).
